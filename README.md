Matt Sagat University of Illinois Chicago CS 342 Project 3

These are the project files for the game of Blackjack. Both the front-end and back-end code is entirely written by me using Java and the JavaFX GUI building library.
This game features a simple artificial intelligence for the dealer. Scroll further for some screenshots highlighting game features.
